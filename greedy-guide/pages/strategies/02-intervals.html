2.2 區間問題全家桶（Interval Scheduling Family）

💡 核心想法

區間貪心的一切糾結都在「按左端點還是右端點排」。記住對應關係：選最多不重疊 → 右端點；完全覆蓋 → 左端點；算最少資源 → 掃描線/小根堆。

🔎 適用特徵
  • 輸入是一堆 [l, r]，或能轉化成區間（如 POJ 1328 的幾何轉化、「站點+半徑」轉化）。
  • 目標是「最多選幾個 / 最少用幾個 / 最少資源數 / 能否覆蓋」。
  • ⚠ 若區間帶權且目標是權和 — 貪心失效，轉 DP（見 2.2.5）。

2.2.1 不重疊區間最多選幾個（活動選擇）— 按右端點排

★★LC 435 無重疊區間區間右端點排序

移除最少的區間，使其餘互不重疊 ⟺ 選出最多互不重疊的區間。

思路

按右端點遞增排序，能選就選（左端點 ≥ 上一個選中的右端點）。證明用領先論證：貪心第 k 個選中區間的右端點 ≤ 任意解第 k 個的右端點（歸納），所以任何解能塞下的，貪心都塞得下。為什麼不是左端點：一個很早開始但很長的區間會堵死後面 — 反例 [1,100],[2,3],[4,5]。

程式碼（C++） 
sort(iv.begin(), iv.end(), [](auto& a, auto& b){ return a[1] < b[1]; });
int keep = 0; long long lastEnd = LLONG_MIN;
for (auto& v : iv)
    if (v[0] >= lastEnd) { keep++; lastEnd = v[1]; }
return iv.size() - keep;

2.2.2 完全覆蓋最少段數 — 按左端點排 + 貪心延伸

★★★LC 1024 視頻拼接區間左端點排序覆蓋

用最少段影片覆蓋 [0, T]。同型：LC 45 跳躍遊戲 II、LC 1326 灌溉花園。

思路

按左端點排序。維護目前已覆蓋到 cur；在所有左端點 ≤ cur 的區間中，選右端點最遠的一段接上（這一步是貪心選擇性質：反正都要有一段跨過 cur，選伸最遠的不會變差），cur 跳到該右端點。跳躍遊戲 II 是同一個模型的陣列形式，可 O(n) 一次掃描。

程式碼（C++，跳躍遊戲 II 形式） 
int jumps = 0, curEnd = 0, farthest = 0;
for (int i = 0; i + 1 < n; i++) {
    farthest = max(farthest, i + nums[i]);
    if (i == curEnd) { jumps++; curEnd = farthest; }  // 必須起跳
}

2.2.3 最少資源數（區間分組）— 掃描線 / 小根堆

★★★LC 253 會議室 II區間堆掃描線

最少需要幾間會議室容納所有會議？同型：洛谷 P2887、CF 手機基地台類。

思路

兩種等價視角：掃描線 — 把開始 +1、結束 −1 排成事件，答案是前綴和最大值（= 最大同時重疊數，這是下界，也構造性地可達）；小根堆 — 按開始時間排序，堆存每間房目前的結束時間，新會議若能接最早空出的房就接，否則開新房。

程式碼（C++，小根堆） 
sort(iv.begin(), iv.end());
priority_queue<int, vector<int>, greater<>> rooms; // 各房結束時間
for (auto& [s, e] : iv) {
    if (!rooms.empty() && rooms.top() <= s) rooms.pop();
    rooms.push(e);
}
return rooms.size();

2.2.4 區間選點 / 最少箭射氣球

★★LC 452 用最少數量的箭引爆氣球區間右端點排序
思路

「最少的點使每個區間至少含一點」— 與活動選擇對偶。按右端點排序，箭射在目前區間的右端點（能捅穿的最晚位置，捅到最多後續區間），左端點超過箭位置就補新箭。答案 = 最多不相交區間數（LP 對偶的離散體現）。

2.2.5 邊界討論：帶權區間 → 貪心失效 → DP

當每個區間帶權 w、目標最大化選中權和：交換論證失敗 — 「結束早」與「權大」不可比。此時按右端點排序後做 dp[i] = max(dp[i−1], w_i + dp[prev(i)])（prev 用二分找），O(n log n)。辨識訓練：看到區間題先問「有沒有權重」，這一問決定貪心/DP 的分岔。
