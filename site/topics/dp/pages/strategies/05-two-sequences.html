2.5 雙序列 DP

🧭 把 dp 表看成「對齊路徑」

在 \((n+1)\times(m+1)\) 的格點圖上，\((i,j)\) 表示兩序列各消耗了 \(i,j\) 個元素。向下代表跳過 \(A_i\)，向右代表跳過 \(B_j\)，斜走代表把兩個元素配對或替換。LCS 是在斜走且字元相等時得 \(1\) 分；編輯距離則讓橫、直、斜三種操作各有成本。這個幾何觀點能直接推導轉移，也能用倒走格點路徑還原一組對齊方案。

問題斜走橫／直走邊界
LCS相等才配對並 +1跳過其中一端任一空序列答案 0
編輯距離相等 \(+0\)，否則替換 \(+1\)插入／刪除 \(+1\)\(\mathrm{dp}[i][0]=i\)、\(\mathrm{dp}[0][j]=j\)
交錯字串不使用從 \(A\) 或 \(B\) 取下一字元由 \(i+j\) 推出第三下標
雙序列計數依配對規則加／乘必須設計去重規則空前綴常是唯一空方案
💡 核心想法

\(\mathrm{dp}[i][j]\) 是「序列 \(A\) 的前 \(i\) 個」與「序列 \(B\) 的前 \(j\) 個」的答案。轉移枚舉兩序列末端字元的處置方式（配對／各自跳過），永遠只有常數種，複雜度 \(O(nm)\)。

🔎 適用特徵
  • 兩條字串/序列之間的比較、對齊、匹配、轉換。
  • 關鍵不變量：處理完前綴 \((i,j)\) 後，剩下的問題只跟後綴有關。
★★LC 1143 最長公共子序列（LCS）雙序列

兩字串的最長公共子序列長度。

思路

若 \(a_i=b_j\)，則 \(\mathrm{dp}[i][j]=\mathrm{dp}[i-1][j-1]+1\)；否則 \(\mathrm{dp}[i][j]=\max\left(\mathrm{dp}[i-1][j],\mathrm{dp}[i][j-1]\right)\)。這是雙序列 DP 的原型。還原方案：從 \(\mathrm{dp}[n][m]\) 倒走，看每步是哪個轉移貢獻的。

程式碼（C++） 
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        dp[i][j] = (a[i - 1] == b[j - 1]) ? dp[i - 1][j - 1] + 1
            : max(dp[i - 1][j], dp[i][j - 1]);
    }
}
★★★洛谷 P1439 排列 LCS雙序列問題轉化

兩個 \(1\) 到 \(n\) 的排列求 LCS，\(n \le 10^5\)——\(O(nm)\) 過不了。

思路

排列的特殊性：元素兩兩相異。把 \(B\) 的每個元素改寫成「它在 \(A\) 中的位置」，LCS 就變成 LIS（公共子序列等價於在兩邊都遞增的位置序列），套 \(O(n\log n)\)。「利用輸入的特殊結構把模型換掉」是 ★★★ 之後的常態。

★★★LC 72 編輯距離雙序列

插入/刪除/替換，把 A 變成 B 的最少操作數。

思路

三種操作對應三個轉移：\(\mathrm{dp}[i][j]=\min\left(\mathrm{dp}[i-1][j]+1,\mathrm{dp}[i][j-1]+1,\mathrm{dp}[i-1][j-1]+[a_i\ne b_j]\right)\)，分別是刪除、插入、替換／免費匹配。邊界為 \(\mathrm{dp}[i][0]=i\)、\(\mathrm{dp}[0][j]=j\)。工業界（diff、拼寫校正）與競賽通吃的經典。

★★★LC 115 不同的子序列雙序列計數

A 的子序列中等於 B 的個數。

思路

計數版雙序列：\(\mathrm{dp}[i][j]\) 是 \(A\) 前 \(i\) 個裡湊出 \(B\) 前 \(j\) 個的方案數，轉移為 \(\mathrm{dp}[i][j]=\mathrm{dp}[i-1][j]+[a_i=b_j]\mathrm{dp}[i-1][j-1]\)。兩項分別代表不用或使用 \(a_i\)。計數 DP 的鐵律：每個方案恰好被一條轉移路徑數到一次——這裡按「\(a_i\) 用不用」分類，天然不重不漏。

★★★LC 44 萬用字元匹配雙序列匹配

模式含 ?（單字元）與 *（任意長度串），判斷全串匹配。

思路

\(\mathrm{dp}[i][j]\) 表示 \(s\) 前 \(i\) 個能否匹配 \(p\) 前 \(j\) 個。* 的轉移為 \(\mathrm{dp}[i][j]=\mathrm{dp}[i][j-1]\lor\mathrm{dp}[i-1][j]\)，兩項分別代表匹配空字串與多吞一個字元。難度再上一級的 LC 10 正則版：x* 是整體單位，轉移分「匹配零次」\(\mathrm{dp}[i][j-2]\) 與「再吞一個」\(\mathrm{dp}[i-1][j]\)（需 \(s_i\) 與 \(x\) 相容）。

程式碼（C++） 
dp[0][0] = true;
for (int j = 1; j <= m; j++) {
    dp[0][j] = dp[0][j - 1] && pattern[j - 1] == '*';
}
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        dp[i][j] =
            (pattern[j - 1] == '*')
                ? dp[i][j - 1] || dp[i - 1][j]
                : dp[i - 1][j - 1] &&
                    (pattern[j - 1] == '?' || pattern[j - 1] == text[i - 1]);
    }
}
★★★LC 97 交錯字串雙序列

C 能否由 A、B 保序交錯而成？

思路

狀態設計練習：\(C\) 的前 \(i+j\) 個由 \(A\) 前 \(i\) 個與 \(B\) 前 \(j\) 個組成——第三條序列的下標是導出量，不用進狀態。轉移為 \(\mathrm{dp}[i][j]=\left(\mathrm{dp}[i-1][j]\land a_i=c_{i+j}\right)\lor\left(\mathrm{dp}[i][j-1]\land b_j=c_{i+j}\right)\)。貪心（誰匹配就先吃誰）在這題有反例，值得親手構造。
