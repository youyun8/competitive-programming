## 1. 理論基礎：字串的共同語言
- 1.1 基本名詞：子串（連續）vs 子序列（不連續）、前綴/後綴/真前綴、字典序
- 1.2 border 與週期：border 定義、週期定義、核心等式 **p 是週期 ⟺ n−p 是 border**
  - Weak Periodicity Lemma：p, q 皆為週期且 p+q ≤ n ⇒ gcd(p,q) 也是週期
  - border 的遞迴結構：s 的所有 border = fail 鏈；border 集合可分成 O(log n) 個等差數列（僅陳述，證明留 2.11）
- 1.3 三種「匹配思維」：滑動比對（KMP/Z）、指紋比對（hash）、自動機（Trie/ACAM/SAM/PAM）
- 1.4 複雜度與資料範圍反推：|s| ≤ 10⁶ 線性家族；子串本質不同計數 → SA/SAM；n ≤ 5000 允許 O(n²) 暴力 DP
- 1.5 實作工具箱：0/1-indexed 慣例、字元集映射 `c-'a'`、多測清空成本、`std::string` 拷貝陷阱、讀入速度

## 2. 策略分類（每章含：核心想法、適用特徵、模板碼、例題卡片）

### 2.1 字串雜湊（Polynomial Hashing）
- 核心：把字串映成數字，O(1) 比較任意子串；「不等必真、相等大概率真」
- 2.1.1 多項式 hash 定義、前綴 hash 表、子串 hash O(1) 公式
- 2.1.2 防卡：自然溢位被 Thue–Morse 卡掉、雙模數 / 隨機 base、生日悖論估碰撞率
- 2.1.3 hash + 二分求最長公共前綴（LCP）→ O(log n) 比較字典序
- 例題：
  - 洛谷 P3370 字串雜湊模板 ★
  - LC 28 找出字串中第一個匹配項 ★（hash 版；KMP 版見 2.2）
  - LC 187 重複的 DNA 序列 ★★
  - CF 1200E Compress Words（前綴 hash 拼接）★★★
  - LC 1044 最長重複子串（二分答案 + hash）★★★★
  - 洛谷 P2852 Milk Patterns（hash 二分做法對照，正解見 2.7）★★★★

### 2.2 KMP、失配函數與週期
- 核心：fail[i] = 前綴 s[1..i] 的最長 border；匹配時失配不回頭
- 2.2.1 fail 陣列的均攤 O(n) 構造與正確性（fail 鏈 = 所有 border）
- 2.2.2 單模式匹配主循環；統計出現次數
- 2.2.3 週期應用：最小週期 = n − fail[n]；整除判斷循環節
- 2.2.4 KMP 自動機（把 fail 轉成 goto 表）→ 銜接 DP 主題的自動機上 DP
- 例題：
  - 洛谷 P3375 KMP 模板 ★★
  - LC 459 重複的子字串（fail 判循環節）★★
  - 洛谷 P4391 無線傳輸 BOI（最小週期，答案 n−fail[n] 的理解題）★★★
  - LC 214 最短迴文串（s + '#' + rev(s) 的 border 技巧）★★★
  - CF 126B Password（既是前綴又是後綴又在中間出現）★★★
  - 洛谷 P3193 GT 考試（KMP 自動機 + 矩陣快速冪，與 DP 主題共用）★★★★★

### 2.3 Z 函數（擴展 KMP）
- 核心：z[i] = s 與 s[i..] 的 LCP 長度；Z-box 維護線性構造
- 2.3.1 Z-box 論證：為何每個位置只擴張一次
- 2.3.2 exkmp：模式串對文本串每個後綴的 LCP
- 2.3.3 Z 與 fail 的互換關係；什麼題 Z 比 KMP 順手（每個位置要「往後配多長」）
- 例題：
  - 洛谷 P5410 Z 函數模板 ★★★
  - CF 432D Prefixes and Suffixes（z + 計數後綴和）★★★★
  - LC 2223 構造字串的總得分和（裸 Z 應用）★★★
  - CF 1968G1 Division + Score（Z + 枚舉，Z 的可加工性示範）★★★★

### 2.4 Manacher（線性迴文半徑）
- 核心：以每個中心的最長迴文半徑 p[i]；利用對稱鏡像避免重複擴張
- 2.4.1 插 '#' 統一奇偶；p[i] 與原串迴文長度的換算
- 2.4.2 「迴文右邊界 R」的單調性 = 均攤線性的來源
- 2.4.3 得到 p[] 之後能回答什麼：最長迴文子串、迴文子串計數、以 i 開頭/結尾的最長迴文
- 例題：
  - 洛谷 P3805 Manacher 模板 ★★★
  - LC 5 最長迴文子串 ★★
  - LC 647 迴文子串數目 ★★
  - 洛谷 P1659 拉拉队（半徑計數 + 貪心取最長）★★★★
  - 洛谷 P4555 最長雙迴文串（前後綴迴文拼接）★★★★

### 2.5 Trie 與 01-Trie
- 核心：字典樹 = 前綴共享的樹；01-Trie = 把數字當 30 位字串，貪心逐位選
- 2.5.1 Trie 基礎：插入/查詢/前綴計數；空間 = 節點數 × 字元集
- 2.5.2 01-Trie 求最大 XOR：從高位貪心走相反位
- 2.5.3 01-Trie 上的刪除與計數（帶 cnt 的節點）
- 2.5.4 可持久化 01-Trie 預告 → 資料結構主題 2.12
- 例題：
  - LC 208 實作 Trie ★
  - 洛谷 P8306 Trie 模板（前綴計數）★★
  - LC 421 陣列中兩數的最大異或值 ★★★
  - 洛谷 P4551 最長異或路徑（樹上前綴 XOR + 01-Trie）★★★
  - CF 888G Xor-MST（Boruvka + 01-Trie，與貪心主題共用）★★★★★

### 2.6 AC 自動機（多模式匹配）
- 核心：Trie + fail 指標 = 在 Trie 上跑 KMP；文本掃一遍同時匹配所有模式串
- 2.6.1 BFS 建 fail；goto 表（Trie 圖）改寫讓轉移 O(1)
- 2.6.2 fail 樹：fail 邊反向成樹，「出現次數」= fail 樹子樹和——從逐點跳 fail O(n√n) 到一次 DFS O(n)
- 2.6.3 ACAM 上的 DP：狀態 =（文本位置, 自動機節點），禁止串計數／必含串計數
- 例題：
  - 洛谷 P3808 AC 自動機（簡單版）★★★
  - 洛谷 P3796 AC 自動機（加強版：哪個模式串出現最多）★★★
  - 洛谷 P5357 AC 自動機（二次加強版：fail 樹子樹和）★★★★
  - 洛谷 P4052 文本生成器（ACAM 上 DP：至少含一個 → 補集）★★★★
  - 洛谷 P2414 阿狸的打字機（fail 樹 DFS 序 + BIT + 離線）★★★★★

### 2.7 後綴數組（Suffix Array）
- 核心：把所有後綴排序；sa/rank 互逆 + height 陣列（相鄰後綴 LCP）
- 2.7.1 倍增構造 O(n log n)（基數排序版）；SA-IS 線性構造只講存在性
- 2.7.2 height 陣列：h[i] ≥ h[i−1]−1 的均攤證明；LCP(sa[i], sa[j]) = min(height[i+1..j]) → ST 表 O(1)
- 2.7.3 應用四件套：不同子串個數 n(n+1)/2 − Σheight、出現 k 次的最長子串（height 分組）、字典序第 k 小後綴、兩串問題（拼接 + 分隔符）
- 例題：
  - 洛谷 P3809 後綴排序模板 ★★★★
  - 洛谷 P4051 字符加密（環形拆倍 + SA）★★★
  - 洛谷 P2408 不同子串個數 ★★★★
  - 洛谷 P2852 Milk Patterns（height 二分/單調佇列）★★★★
  - 洛谷 P4248 差異（Σ LCP 的 height 單調棧計數）★★★★★

### 2.8 後綴自動機（SAM）
- 核心：接受所有後綴的最小 DFA；每個節點 = 一個 endpos 等價類 = 一段連續長度的子串
- 2.8.1 endpos 等價類三性質：包含關係成樹（parent 樹）、長度連續、總節點 ≤ 2n−1
- 2.8.2 線上增量構造（實作背下來，論證講 clone 的意義）
- 2.8.3 應用地圖：本質不同子串數 Σ(len−len[link])、出現次數 = parent 樹子樹 cnt 和、最長公共子串（在 SAM 上走）、字典序第 k 小子串
- 2.8.4 SAM vs SA 選型：要 endpos 結構/線上 → SAM；要 height/字典序區間 → SA
- 例題：
  - 洛谷 P3804 SAM 模板（出現次數×長度 最大值）★★★★★
  - 不同子串個數（SAM 版，對照 2.7）★★★★
  - SPOJ LCS 兩串最長公共子串 ★★★★
  - CF 235C Cyclical Quest（環形模式在 SAM 上匹配）★★★★★
  - 洛谷 P3975 弦論（第 k 小子串，兩種語意）★★★★★

### 2.9 迴文自動機（PAM / 迴文樹）
- 核心：每個節點 = 一個本質不同迴文子串；兩棵根（長 0 / 長 −1）；fail = 最長迴文真後綴
- 2.9.1 增量構造：找「x 使 xAx 是迴文」的跳 fail 過程、均攤線性
- 2.9.2 直接讀出：本質不同迴文子串數 = 節點數、每個迴文的出現次數（fail 樹子樹和）
- 2.9.3 與 Manacher 分工：只要最長/計數 → Manacher；要對每個迴文「作為對象」統計 → PAM
- 例題：
  - 洛谷 P5496 迴文自動機模板 ★★★★
  - 洛谷 P3649 [APIO2014] 回文串（出現次數 × 長度最大值）★★★★
  - 洛谷 P4287 雙倍迴文（fail 樹 + 長度一半指標 trans）★★★★★
  - CF 906E Reverses（PAM 上 DP，提及）★★★★★

### 2.10 最小表示法與 Lyndon 分解
- 核心：環同構的正規化（最小表示）；s = w₁w₂…wₖ、wᵢ 為字典序遞減 Lyndon 串的唯一分解
- 2.10.1 最小表示法雙指標 O(n) 論證
- 2.10.2 Duval 演算法 O(n) 求 Lyndon 分解；最後一個分解塊 = 最小後綴
- 2.10.3 應用：字串拼接的最小字典序、最小後綴/最大後綴、significant rotations
- 例題：
  - 洛谷 P1368 最小表示法模板 ★★★
  - LC 899 有序隊列（k=1 退化成最小表示）★★★
  - 洛谷 P6114 Lyndon 分解模板 ★★★★
  - CF 496E? 不收；改收 UVA 719 Glass Beads（最小表示或 SA）★★★

### 2.11 專家專題（★★★★★）
- 2.11.1 失配樹（border 樹）：fail 指標反向成樹，「兩前綴的最長公共 border」= 樹上 LCA——洛谷 P5829 ★★★★
- 2.11.2 廣義 SAM：多串共建，離線 Trie 上 BFS 構造——洛谷 P6139 模板 ★★★★★
- 2.11.3 SAM/PAM × 資料結構：parent 樹上線段樹合併維護 endpos——CF 1037H Security（SAM + 線段樹合併貪心走字典序）★★★★★
- 2.11.4 後綴樹視角：反串 SAM 的 parent 樹 = 後綴樹；Ukkonen 線上構造只講直覺
- 2.11.5 border 理論深入：border 等差數列結構 + 週期引理證明；CF 1286E?（提及）；「所有前綴的最小週期」類題
- 2.11.6 位平行匹配：bitset / Shift-And，O(nm/64) 暴力的實戰價值

## 3. 常見錯誤與陷阱
- hash 被卡全家桶：自然溢位 mod 2⁶⁴ 被 Thue–Morse 構造卡、單模數生日悖論、base 未隨機
- 「KMP 是 O(n)」的誤解：寫成每次失配從頭比 → O(nm)；fail 鏈上跳的均攤論證錯位
- Z 函數與 fail 混用：z[i] 是「從 i 開始」、fail[i] 是「以 i 結尾」——方向相反
- SA 三件套變數混淆：sa（排名→位置）、rank（位置→排名）、height 下標偏移
- SAM 拓撲序錯誤：用 fail 鏈 DFS 統計 cnt 時未按 len 排序（基數排序 vs 建圖 DFS）
- PAM/Manacher 忘記奇偶：Manacher 忘插分隔符、PAM 的 −1 根語意
- 多測清空成本：Trie/ACAM 用 memset 整包清 → O(T·SIZE) TLE；要按用過的節點清
- 字元集陷阱：題目說小寫字母結果有大寫/數字；`c-'a'` 越界不報錯
- 爆 int：hash 乘法、SAM 的 cnt×len、迴文計數 n(n+1)/2
