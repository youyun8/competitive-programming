6學習路線圖

🎯 路線圖的使用方式

字串的特殊之處：模板佔比極高，但「背得出」與「用得對」之間隔著 border/週期這層理論。每個階段都遵循「讀懂 → 默寫模板 → 過對拍 → 講出線性複雜度的論證 → 變形題」五步。星級只是參考，畢業檢定過不了就留在原階段補洞；搭配側欄的全站題目總表按標籤篩選刷題。

第一階段 ★–★★：四件輕武器（約 2 週）

目標：hash、KMP、Trie、Manacher 模板能默寫並通過對拍；懂 border ⟺ 週期。

  • hash：P3370 → LC 187 → LC 28（暴力/hash/KMP 三寫）
  • KMP：P3375 → LC 459 → P4391（週期兩兄弟）
  • Trie：LC 208 → P8306（多測清空的第一課）
  • Manacher：LC 5（先中心擴展再 Manacher）→ LC 647

✅ 畢業檢定：LC 28 三種寫法各自省掉了哪些重複比較，講給別人聽；並證明最小週期 = n − fail[n]。

第二階段 ★★★：組合技與自動機入門（約 1 個月）

目標：Z 函數、ACAM、最小表示法上手；「拼接+分隔符」「二分+hash」「fail 樹子樹和」三個句型成為反射。

  • Z：P5410 → LC 2223 → CF 432D
  • KMP 應用：LC 214（拼接技巧）→ CF 126B（fail 鏈枚舉 border）
  • ACAM：P3808 → P3796 → P5357（見證跳 fail → fail 樹的升級）
  • hash 組合：CF 1200E → LC 1044 → P2852（hash 版）
  • 旋轉：P1368 → LC 899 → UVa 719
  • 01-Trie：LC 421 → P4551

✅ 畢業檢定：獨立完成 P5357（fail 樹子樹和），並解釋為什麼逐位置跳 fail 會超時。

第三階段 ★★★★：後綴結構（約 1–2 月）

目標：SA、SAM、PAM 三大重型結構的模板與應用地圖；同一題能說出兩種以上結構的做法。

  • SA：P3809 → P4051 → P2408 → P2852（SA 版，與 hash 版對照）
  • SAM：P3804 → 不同子串數（SAM 版）→ SPOJ LCS
  • PAM：P5496 → P3649（與 P3804 對照「打點+子樹和」）
  • Manacher 進階：P1659 → P4555
  • ACAM 上 DP：P4052；失配樹：P5829

✅ 畢業檢定：P2852 用 SA 與 SAM 各寫一遍全 AC；畫出 banana 的 SAM（含 clone）與 parent 樹。

第四階段 ★★★★★：結構 × 結構（長期）

目標：廣義 SAM、SAM×線段樹合併、border 理論；面對 CF 2400+ 字串題有系統的切入流程。

  • 廣義 SAM：P6139；SAM 進階：CF 235C → P3975
  • SAM × 資料結構：CF 1037H（先補資料結構主題的線段樹合併）
  • ACAM 壓軸：P2414 阿狸的打字機
  • SA 壓軸：P4248 差異；PAM 壓軸：P4287 → CF 906E（讀懂框架）
  • 理論：Periodicity Lemma 證明、border 等差結構、Duval：P6114

✅ 畢業檢定：拿到一道陌生 CF 2400+ 字串題，能在 10 分鐘內完成「問題族分類 → 候選工具 → 複雜度預算」的書面分析（不必解出）。
