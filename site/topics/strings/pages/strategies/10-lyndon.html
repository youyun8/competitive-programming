2.10 最小表示法與 Lyndon 分解

💡 核心想法

兩個關於「旋轉與字典序」的線性工具。① 最小表示法：在 \(s\) 的所有旋轉中找字典序最小者——雙指標互相淘汰，\(O(n)\)。② Lyndon 分解：Lyndon 串 = 嚴格小於自己所有真後綴的串；任何 \(s\) 都能唯一分解為 \(s=w_1w_2\cdots w_k\)，其中 \(w_1\ge w_2\ge\cdots\ge w_k\) 皆為 Lyndon 串；Duval 演算法 \(O(n)\) 求出。

🔁 最小表示法的雙指標為何正確？

維護兩個候選起點 \(i,j\)，比較 \(s_{i+k}\) 與 \(s_{j+k}\)：若 \(s_{i+k}>s_{j+k}\)，則以 \(i,i+1,\dots,i+k\) 為起點的旋轉全部輸給對應的 \(j\) 系旋轉（前 \(k\) 字元相同、第 \(k+1\) 字元見分曉）——一次淘汰 \(k+1\) 個候選，\(i\) 直接跳到 \(i+k+1\)。兩指標合計最多前進 \(2n\)，線性。「一次比較、批量淘汰」是雙指標論證的經典款。

🔎 適用特徵
  • 環同構判定／正規化（項鍊、地圖旋轉等價）→ 最小表示。
  • 「最小/最大後綴」「無限重複後比大小」→ Lyndon／Duval 的直接產物。
  • 需要所有旋轉的完整排序而非只要最小 → 回 SA 拆環倍長。
模板（C++）：最小表示法 + Duval 
int minRotation(const string& s) {          // 回傳最小旋轉的起點
    int n = s.size(), i = 0, j = 1, k = 0;
    while (i < n && j < n && k < n) {
        char a = s[(i + k) % n], b = s[(j + k) % n];
        if (a == b) { k++; continue; }
        (a > b ? i : j) += k + 1;           // 批量淘汰
        if (i == j) j++;
        k = 0;
    }
    return min(i, j);
}
void duval(const string& s) {               // 輸出每個 Lyndon 塊的右端
    int n = s.size(), i = 0;
    while (i < n) {
        int j = i + 1, k = i;               // 不變量：s[i..k] 是 s[i..j-1] 的
        while (j < n && s[k] <= s[j])       // 週期性延伸（w^m + w 的前綴）
            k = (s[k] < s[j]) ? i : k + 1, j++;
        while (i <= k) report(i + j - k - 1), i += j - k;  // 切出整塊 w
    }
}
★★★洛谷 P1368 工藝／最小表示法模板最小表示

求數列所有旋轉中字典序最小者。

思路

裸模板；注意元素是整數不是字元（比較用值）。寫完後自我測試：全相同元素（aaaa）會不會死循環？——k<n 的終止條件就是為它準備的。

★★★LC 899 有序隊列最小表示觀察

每次可把前 \(k\) 個字元中任一個移到末尾，求能得到的最小字串。

思路

分水嶺觀察：\(k\ge 2\) 時操作等價於任意交換相鄰（可以冒泡排序）→ 直接排序；\(k=1\) 時只能旋轉 → 最小表示法。字串題常見的「操作能力分級」：先分析操作的生成能力，再選工具。

★★★UVa 719 Glass Beads最小表示

項鍊剪開成鏈，求字典序最小的剪法（輸出起點）。

思路

最小表示法的具象化敘事版。多測 + 要求「最小起點下標」——若最小旋轉不唯一要回傳最小的 \(i\)，模板回傳 min(i,j) 恰好處理了這點（想想為什麼）。

★★★★洛谷 P6114 Lyndon 分解模板Lyndon

輸出 Lyndon 分解所有分割點的 XOR。

思路

Duval 三指標的正確性靠不變量：\(s_{i..j-1}\) 恆為「某 Lyndon 串 \(w\) 的若干次重複接上 \(w\) 的一個前綴」。讀入 \(5\times 10^6\)——線性與 IO 都得認真。附贈結論：最後一個 Lyndon 塊 = s 的最小後綴；對每個前綴求最小後綴也能由 Duval 邊跑邊答。

⏫ 進階去向

Lyndon 理論再往上是 runs 定理（本原平方串、字串中 run 的個數 \(< n\)）與 Lyndon 樹，屬於 ★★★★★ 理論題的軍火庫（2.11.5 提及）；競賽實用度最高的仍是「最小後綴／最小旋轉」這兩個直接產物。
