2.6 線段樹基礎（build／懶標記／資訊設計）

💡 核心想法

把 \([1,n]\) 遞迴對半切成一棵樹，任意區間可拆成 \(O(\log n)\) 個節點——只要資訊可合併（1.2），就能單點改 \(O(\log n)\)、區間查 \(O(\log n)\)。懶標記解鎖區間修改：整段覆蓋的節點「先記帳、不下推」，之後路過時才 pushdown——正確性的關鍵是標記可複合（兩次記帳能合成一次）且標記對資訊的影響可計算（區間加 \(v\) 對和的影響 = \(v\times\)段長）。

🔎 適用特徵
  • 修改與查詢交錯、資訊可合併但不可差分（min/max/最大子段和）——BIT 管不了的都歸它。
  • 區間修改（加、乘、賦值、取 min…）+ 區間查詢。
  • 資訊不可合併（眾數、中位數）→ 別硬上，去 2.10。
模板（C++）：區間加 + 區間和（P3372） 
#define ls (u << 1)
#define rs (u << 1 | 1)
long long sum[N << 2], tag[N << 2];      // 空間 4n！
void pushup(int u) { sum[u] = sum[ls] + sum[rs]; }
void apply(int u, int len, long long v) { sum[u] += v * len; tag[u] += v; }
void pushdown(int u, int len) {
    if (!tag[u]) return;
    apply(ls, len - len / 2, tag[u]);
    apply(rs, len / 2, tag[u]);
    tag[u] = 0;
}
void update(int u, int l, int r, int ql, int qr, long long v) {
    if (ql <= l && r <= qr) { apply(u, r - l + 1, v); return; }
    int m = (l + r) >> 1; pushdown(u, r - l + 1);
    if (ql <= m) update(ls, l, m, ql, qr, v);
    if (qr > m)  update(rs, m + 1, r, ql, qr, v);
    pushup(u);
}
long long query(int u, int l, int r, int ql, int qr) {
    if (ql <= l && r <= qr) return sum[u];
    int m = (l + r) >> 1; pushdown(u, r - l + 1);
    long long s = 0;
    if (ql <= m) s += query(ls, l, m, ql, qr);
    if (qr > m)  s += query(rs, m + 1, r, ql, qr);
    return s;
}
★★★洛谷 P3372 線段樹模板一線段樹懶標記

區間加、區間和。

思路

懶標記的第一課。寫的時候給自己立規矩：apply（打標記）／pushdown（下傳）／pushup（上收）三個函式分工明確，update/query 只負責遞迴骨架——之後所有線段樹變形都只改 apply 和 pushup。

★★洛谷 P1531 I Hate It線段樹

單點改、區間 max。LC 307（單點改區間和）同級。

思路

無懶標記的入門版：先把「骨架」寫熟再上標記。順便確認：max 不可差分，BIT 做不了這題（存在魔改 BIT 但複雜度退化）——這就是線段樹的生態位。

★★★★洛谷 P3373 線段樹模板二線段樹多重標記

區間加、區間乘、區間和（mod p）。

思路

多重標記複合的教科書：把標記統一成仿射變換 \(x\mapsto ax+b\)——乘 \(m\)：\((a,b)\to(ma, mb)\)；加 \(v\)：\((a,b)\to(a, b+v)\)。複合有順序（先乘後加），pushdown 一次下傳兩個。搞懂「標記是函數、複合是函數合成」，任何新標記組合都能自己推。

★★★★洛谷 P4513 小白逛公園線段樹資訊設計

單點改、查區間最大子段和。

思路

最大子段和不可直接合併——加維修複：每節點存四元組（總和、最大前綴、最大後綴、最大子段），合併時跨中線的候選 = 左的後綴 + 右的前綴。這就是 1.2 說的「設計複合資訊把不可合併變可合併」，是線段樹從模板走向創作的分水嶺。SPOJ GSS 系列是它的連續劇。

⚠ 高頻坑

① 空間 4n（遞迴切不齊時樹高多一層），開 2n 會 RE 在第 7 個點。② pushdown 忘記清 tag、apply 忘記同時改 sum 和 tag。③ 先乘後加的順序寫反。④ 區間長度奇偶：len - len/2 給左兒子。完整清單見第 3 章。

⏫ 進階去向

下標從「位置」換成「值域」→ 權值線段樹；不先建滿 → 動態開點；兩棵樹合併 → 線段樹合併；插入直線 → 李超樹——全在 2.7。掃描線的「覆蓋計數不下傳」特技在 2.8。
