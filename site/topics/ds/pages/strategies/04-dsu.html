2.4 並查集（DSU）

💡 核心想法

維護「不相交集合」的兩個操作：find（我屬於哪個等價類）與 union（合併兩類）。路徑壓縮 + 按秩（大小）合併後均攤 \(O(\alpha(n))\)≈常數。等價類的語意可以很寬：連通性、朋友圈、「這些變數相等」、「這一段已被佔用」——凡是自反、對稱、傳遞的關係，都可以丟給 DSU。

🔎 適用特徵
  • 只合併、不分裂（分裂 → 離線倒序或 LCT）。
  • 「加邊後問連通」「分組」「傳遞約束（相等/同組）」。
  • 帶「關係量」的約束（距離、奇偶、mod k 類別）→ 帶權/種類 DSU。
模板（C++）：路徑壓縮 + 按大小合併 
int fa[N], sz[N];
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }
bool unite(int x, int y) {
    x = find(x); y = find(y);
    if (x == y) return false;
    if (sz[x] < sz[y]) swap(x, y);
    fa[y] = x; sz[x] += sz[y];
    return true;
}
// 帶權版：w[x] = x 到父節點的關係量，find 時沿路累積並壓縮
int find(int x) {
    if (fa[x] == x) return x;
    int r = find(fa[x]);
    w[x] += w[fa[x]];          // 關係量按語意複合（加法/異或/mod）
    return fa[x] = r;
}
★洛谷 P3367 並查集模板DSU

合併與查詢是否同集合。P1551 親戚 ★ 同款敘事版。

思路

模板。兩個優化各自就夠 \(O(\log n)\)，合用近常數；競賽寫「路徑壓縮 + 按大小」最順手（size 常有語意）。

★★洛谷 P1892 團伙DSU種類

朋友的朋友是朋友；敵人的敵人是朋友。問最多幾個團伙。

思路

種類（拆點）DSU：每人拆成「本體 \(x\)」與「敵人陣營 \(x+n\)」；\(x\) 與 \(y\) 為敵 → 合併 \((x, y+n)\) 與 \((y, x+n)\)。「用拆點編碼關係」是 DSU 最重要的變形之一，食物鏈是它的三倍點升級。

★★★洛谷 P1197 [JSOI2008] 星球大戰DSU離線倒序

依序摧毀節點，每次摧毀後問連通塊數。

思路

DSU 不會刪——時間倒流：從全毀狀態開始，倒序「復活」節點並加回相鄰邊，連通塊數的變化反著記錄。離線思維的第一課（1.4）：把刪除變成插入。

★★★洛谷 P1955 [NOI2015] 程序自動分析DSU離散化

一堆 \(x_i=x_j\) 與 \(x_i\ne x_j\) 約束，判可滿足性。

思路

等式是等價關係 → 先把所有等式 union，再逐條檢查不等式兩端是否同根。兩個要點：等式全部優先處理（順序自由是因為約束無方向）；值域 \(10^9\) → 離散化。DSU 判可滿足性的原型題。

★★★★洛谷 P2024 [NOI2001] 食物鏈DSU帶權

A 吃 B、B 吃 C、C 吃 A 的三類動物，判斷 k 句話中幾句是假話。

思路

兩種等價解法：拆三倍點（x 的「同類/獵物/天敵」三分身，每句話合併三對）或帶權 DSU（w[x] = 與根的關係 mod 3，find 時累加、union 時解方程）。帶權版更通用（奇偶、距離、異或關係全是同一模板換運算），值得逼自己寫這版。

⏫ 進階去向

可撤銷 DSU：不壓縮、只按秩，用棧記錄修改逐步回滾——單獨沒用（只能撤最近的），配上線段樹分治（2.13.2）就能處理「邊有存活時間區間」的離線動態連通性；真·強制線上動態連通性才需要 LCT（2.13.3）。另見貪心主題 2.9 的「DSU 找下一個空位」——把 fa 指標當跳轉表用的妙招。
