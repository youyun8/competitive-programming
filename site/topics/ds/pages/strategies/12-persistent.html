2.12 可持久化資料結構

💡 核心想法

修改時不覆蓋舊節點：沿修改路徑複製 \(O(\log n)\) 個新節點、其餘子樹與舊版本共享——每個版本只多花 \(O(\log n)\) 空間，所有歷史版本永久可查。殺手級應用是主席樹（可持久化權值線段樹）：對每個前綴 \(1..i\) 存一個版本，「區間 \([l,r]\) 的值域資訊」＝版本 r 減版本 l−1（權值樹逐節點可減！），區間第 k 小就在「差樹」上做樹上二分。

🔎 適用特徵
  • 「查詢歷史版本」「回到第 i 次操作之後」→ 可持久化陣列/線段樹。
  • 「區間第 k 小／區間值域統計」且強制線上 → 主席樹（能離線 → 整體二分也行，見 2.8.4）。
  • 「區間內與 x 的最大 XOR」→ 可持久化 01-Trie（同款前綴相減）。
模板（C++)：主席樹（靜態區間第 k 小） 
int rt[N], ls[M], rs[M], cnt[M], tot;      // M = n * (logV + 1) 左右
int clone(int u) {
    int v = ++tot;
    ls[v] = ls[u];
    rs[v] = rs[u];
    cnt[v] = cnt[u];
    return v;
}
int insert(int u, int l, int r, int x) {   // 回傳新版本的根
    u = clone(u);
    cnt[u]++;
    if (l == r) {
        return u;
    }
    int m = (l + r) >> 1;
    if (x <= m) {
        ls[u] = insert(ls[u], l, m, x);
    } else {
        rs[u] = insert(rs[u], m + 1, r, x);
    }
    return u;
}
int kth(int a, int b, int l, int r, int k) {  // 版本 b 減版本 a：樹上二分
    if (l == r) {
        return l;
    }
    int m = (l + r) >> 1, c = cnt[ls[b]] - cnt[ls[a]];
    return k <= c ? kth(ls[a], ls[b], l, m, k)
                  : kth(rs[a], rs[b], m + 1, r, k - c);
}
// build：rt[i] = insert(rt[i-1], 1, V, a[i])
// query(l, r, k)：kth(rt[l-1], rt[r], 1, V, k)
★★★洛谷 P3919 可持久化陣列模板可持久化

在某歷史版本上單點改（產生新版本）或單點查。

思路

可持久化的最小可行品：線段樹當陣列用，改一次複製一條根到葉的路徑。先寫它搞懂「共享子樹」的指標關係，再上主席樹就只剩「權值化 + 相減」兩步。

★★★★洛谷 P3834 靜態區間第 k 小（主席樹模板）主席樹

\(2\times 10^5\) 次詢問區間第 k 小，強制線上不限。

思路

離散化 → 逐前綴 insert 建版本 → 詢問用兩版本相減做樹上二分。三個理解檢查點：為什麼權值樹可以相減（cnt 是加性資訊）；為什麼樹上二分要同步走兩棵樹；空間為什麼是 \(O(n\log n)\)。這題就是「可持久化」在簡歷上的代表作。

★★★★洛谷 P4735 最大異或和可持久化Trie

序列尾部追加；查詢 \([l,r]\) 內某後綴 XOR 與給定值的最大異或。

思路

前綴 XOR 轉化後變成「在版本區間裡找與 x 最大異或」：可持久化 01-Trie，每個節點存「此前綴下有幾個數經過」，查詢時用 cnt[版本r] − cnt[版本l−1] > 0 判斷該分支在區間內是否存在，貪心走相反位。01-Trie 本體見字串主題 2.5；「前綴版本相減」與主席樹一字不差。

★★★★★洛谷 P2617 Dynamic Rankings樹套樹BIT套主席樹

帶單點修改的區間第 k 小。

思路

修改破壞「前綴版本」結構——改用 BIT 套權值線段樹：BIT 的每個節點掛一棵動態開點權值樹管它的管轄段；查詢時同時抓 \(O(\log n)\) 棵「加樹」與「減樹」一起樹上二分。單次 \(O(\log^2 n)\)、空間 \(O(n\log^2 n)\)。離線的話整體二分（2.8.4）更好寫——線上/離線的價差在這題最直觀。

⚠ 空間紀律（又一次）

① 主席樹 M = \(n(\lceil\log V\rceil+1)\)；帶修樹套樹 M = \(O(n\log^2)\)——先算再開，可持久化的 MLE 全是空間拍腦袋。② 另外 clone 出來的節點別忘了它是「值拷貝」：修改 clone 前的舊指標是經典懸案級 bug。
