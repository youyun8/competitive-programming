2.5 樹狀數組（BIT / Fenwick Tree）

💡 核心想法

c[i] 管轄 \((i-\mathrm{lowbit}(i),\ i]\) 這段——沿 i += lowbit(i) 往上打點、沿 i -= lowbit(i) 往下收前綴，兩個方向各 \(O(\log n)\)。BIT 只會一件事：可差分資訊的前綴查詢（和、計數）；但它碼量 8 行、常數是線段樹的 1/4～1/8，能用 BIT 的場合用線段樹是浪費生命。

🔎 適用特徵
  • 單點改 + 區間和（差分後也涵蓋：區間加+單點查、區間加+區間和）。
  • 權值 BIT：把值域當下標——動態計數「≤ x 有幾個」＝逆序對、名次、第 k 小。
  • 資訊必須可差分：\(\mathrm{sum}(l,r)=S(r)-S(l-1)\)。min/max 不行 → 線段樹。
模板（C++）：BIT + 上面的三個變形 
int tree[N], n;
void add(int pos, int value) {
    for (; pos <= n; pos += pos & -pos) {
        tree[pos] += value;
    }
}
int qry(int pos) {
    int sum = 0;
    for (; pos; pos -= pos & -pos) {
        sum += tree[pos];
    }
    return sum;
}
// 區間加 + 單點查：對差分陣列開 BIT，add(l,v), add(r+1,-v)，查 = qry(i)
// 區間加 + 區間和：兩個 BIT 維護 d[i] 與 i*d[i]，
//   sum(1,r) = (r+1)*qry1(r) - qry2(r)
// BIT 上倍增求第 k 小（權值 BIT）：O(log n)，比二分套查詢省一個 log
int kth(int k) {
    int pos = 0;
    for (int j = __lg(n); j >= 0; j--) {
        if (pos + (1 << j) <= n && tree[pos + (1 << j)] < k) {
            pos += 1 << j;
            k -= tree[pos];
        }
    }
    return pos + 1;
}
★★洛谷 P3374 BIT 模板一BIT

單點加、區間和。P3368 模板二（區間加、單點查）★★ 是差分版。

思路

八行模板。兩個模板連做，體會「BIT 不變，變的是餵給它的陣列（原陣列 vs 差分陣列）」——這個觀點讓「區間加區間和」的雙 BIT 推導變得自然。

★★洛谷 P1908 逆序對BIT權值離散化

數 \(i<j\) 且 \(a_i>a_j\) 的對數。

思路

權值 BIT 第一課：從左往右掃，每個 \(a_j\) 查「已插入且 > \(a_j\)」的個數（= 已插入數 − \(qry(a_j)\)），再插入。值域大先離散化。歸併排序解法對照：同一個「跨越中線的貢獻」兩種算法。

★★★LC 315 計算右側小於當前元素的個數BIT權值

逆序對的「每個位置各自報數」版。

思路

倒序掃 + 權值 BIT 查 \(qry(a_i-1)\)。跟 P1908 湊成一對：正掃查大的、倒掃查小的——方向與不等號的對應關係自己推一次。

★★★★洛谷 P5094 [USACO04OPEN] MooFest 加強版BIT貢獻拆解

求 \(\sum_{i<j}\max(v_i,v_j)\cdot|x_i-x_j|\)。

思路

按 \(v\) 排序讓 max 消失（後加入者的 \(v\) 就是 max），剩下 \(\sum v_j\cdot|x_j - x_i|\)：開兩個權值 BIT（按座標的個數、座標和），絕對值拆成「左邊的：\(cnt\cdot x_j - sumx\)」與「右邊的：\(sumx - cnt\cdot x_j\)」。排序消 max + 雙 BIT 拆絕對值是貢獻類計數的標準連招。

★★★★BIT 上倍增求第 k 小（技巧條目）BIT倍增

權值 BIT 支援「插入/刪除 + 查全域第 k 小」，單次 \(O(\log n)\)。

思路

從高位往低位試探：如果跨出 \(2^j\) 步後累計個數仍 < k，就真的跨出去並扣除。它讓「值域固定的平衡樹需求」（P3369 的弱化版！）用 8+7 行搞定——寫平衡樹之前先問自己：值域能離散化嗎？能的話 BIT 倍增就夠。P1801 黑匣子可用此解對照對頂堆。

⏫ 進階去向

資訊不可差分（min/max）、修改是「區間賦值/取min」→ 線段樹；二維數點的離線引擎就是權值 BIT（2.8.2）；「BIT 套主席樹」處理帶修第 k 小（2.12.4）。
